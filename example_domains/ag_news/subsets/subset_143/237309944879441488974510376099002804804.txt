MacCentral - Later this week Intuit Inc. will officially announce QuickBooks: Pro 2005 for Mac, the latest release of its small business accounting software -- although the company has already posted details of the new release to its Web site. Major changes in this new release include integrated iCal support, the ability to back up data files to .Mac, easier file sharing with the Windows version of QuickBooks and the ability to add PDF backgrounds to forms.
